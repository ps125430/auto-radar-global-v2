# Auto Radar 北極星儀表板 v1.0

供每日使用的知識庫唯讀任務中心。

## 資訊架構

首頁依序呈現：

- 北極星、今日航向、前三機會與禁航區。
- 全球洋流圖、潮汐、市場狀態與資料健康度。
- 聰明資金、籌碼擁擠、市場能量與資金速度。
- 知識庫、每日學習、知識演化與決策檢討。

所有卡片共用標題、分數、趨勢、觀察窗、原因與證據結構。缺少核准資料時顯示等待狀態，不以示意值代替正式內容。

## 更新資料

```powershell
python Scripts/Dashboard/build_dashboard_data.py
```

## 開啟

以瀏覽器開啟 `Dashboard/index.html`。產生的資料位於
`Dashboard/dashboard-data.js`，不依賴正式執行層服務。

## 邊界

儀表板不計算機會分數、信心度、策略、市場狀態、資金流或交易行動。
尚未建立的核准紀錄會顯示為等待資料。
