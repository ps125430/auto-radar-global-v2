# Knowledge Core

This directory is the formal repository storage layer for Auto Radar learning assets.

It is different from `Docs/KNOWLEDGE_CORE.md`:

* `Docs/KNOWLEDGE_CORE.md` defines the concept and structure.
* `Knowledge/` stores actual research records, lessons, playbooks, hypotheses, and decision logs.

## Directory Structure

* `Events/` - market event records and event impact notes
* `Lessons/` - lessons created from outcome review and Root Cause Analysis
* `Playbooks/` - validated repeatable market playbooks
* `Hypothesis/` - unvalidated research hypotheses
* `DecisionLogs/` - decision records with evidence and outcomes
