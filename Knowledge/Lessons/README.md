# Lessons

Purpose:

Store lessons created after decisions are reviewed against market outcomes.

This folder is used for:

* post-decision review
* Root Cause Analysis
* false positive review
* false negative review
* missed risk identification
* model improvement notes

Each lesson should preserve:

* related decision or event
* expected outcome
* actual outcome
* root cause
* what Auto Radar missed
* what should change
* whether model change is approved

Rules:

* Lessons do not automatically change production scoring.
* Lessons may create hypotheses or playbook candidates.
* 小P must approve any model or scoring adjustment before 小C implements it.
