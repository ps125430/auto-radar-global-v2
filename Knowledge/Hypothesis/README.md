# Hypothesis

Purpose:

Store research ideas that are not yet validated for production use.

This folder is used for:

* market learning hypotheses
* score modifier candidates
* early pattern observations
* Market Evolution Framework cases
* research-only ideas before model approval

Each hypothesis should preserve:

* hypothesis name
* source
* research owner
* evidence summary
* test method
* required data
* validation status
* promotion criteria
* rejection criteria

Rules:

* Hypotheses are not production rules.
* Hypotheses must not directly change Theme Score, Stage Score, or Decision Score.
* Validated hypotheses may be promoted only after review by 小P.
* 小C implements only approved specifications.
