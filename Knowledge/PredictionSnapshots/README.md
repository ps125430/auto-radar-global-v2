# Prediction Snapshots

Purpose:

Store templates and future prediction snapshots captured before important market events.

A prediction snapshot freezes what Auto Radar believed before an event outcome is known. This prevents hindsight bias during Decision Lab validation.

This folder is for validation memory only. It does not directly change Theme Score, Stage Score, Decision Score, Dashboard behavior, pipeline behavior, or production trading logic.

Rules:

* Capture snapshots before the event result is known.
* Separate pre-event evidence from post-event outcome.
* Link snapshots to Case Library and Decision Lab when a formal case is opened.
* Do not convert a single snapshot into a production rule.

Current status:

Template only. No real prediction snapshot data has been added.
