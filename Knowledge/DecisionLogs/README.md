# DecisionLogs

Purpose:

Store Auto Radar decision records with evidence, final action, and outcome review.

This folder is used for:

* daily decision logs
* evidence snapshots
* Watch / Wait / Avoid / Buy records
* 綠茶 final action records
* outcome tracking
* later Root Cause Analysis

Each decision log should preserve:

* decision date
* market mode
* attack index
* main theme
* next theme
* avoid themes
* suggested exposure
* entry condition
* stop loss condition
* take profit condition
* confidence score
* evidence summary
* final action by 綠茶
* outcome
* review status

Rules:

* Every decision should leave evidence.
* Every failed decision should be reviewable.
* Decision logs are records, not automatic future instructions.
* Outcome review may create Lessons or Hypothesis entries.
