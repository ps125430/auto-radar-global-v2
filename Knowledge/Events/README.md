# Events

Purpose:

Store structured market event records for Auto Radar research and future market learning.

This folder is used for:

* major macro events
* earnings events
* policy events
* industry catalysts
* theme-triggering news
* event-to-theme-to-leader causal chains

Each event record should preserve:

* event date
* event name
* market expectation before the event
* actual result
* surprise level
* direct and indirect affected themes
* core leaders
* market reaction
* evidence notes
* follow-up outcome

Rules:

* Event records are research evidence, not trade instructions.
* Single events must not create automatic Buy decisions.
* Event conclusions must remain traceable to evidence.
