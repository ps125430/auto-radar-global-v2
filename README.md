# Auto Radar Global v2

Auto Radar 是一套可追溯、可驗證、會透過 Outcome 與 Review 累積知識的 Adaptive Decision Intelligence System。

它不是股票網站，也不是券商 APP。產品目標是讓使用者每天在 30 秒內理解：

- 今天怎麼做。
- 今天關注什麼。
- 今天避免什麼。
- 為什麼。
