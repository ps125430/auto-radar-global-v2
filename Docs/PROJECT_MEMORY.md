# Project Mission

Auto Radar Global v2 is an Adaptive Decision Intelligence System for market research, decision validation, and evidence-based learning.

The project does not aim to predict every market move. It aims to preserve evidence, review decisions, learn from outcomes, and improve future decision quality.

# Why Auto Radar Exists

Auto Radar exists because market decisions must not disappear into chat history, memory drift, or unverified intuition.

Every meaningful decision should leave:

* evidence
* context
* assumptions
* outcome
* review
* lesson
* candidate improvement

# Core Philosophy

The core philosophy is:

* every decision leaves evidence
* every failure has a root cause
* every correction becomes knowledge
* every knowledge asset remains traceable
* no unvalidated idea becomes a production rule

# Confirmed Principles

* GitHub is the only official knowledge base.
* Chat history is not an official record.
* PROJECT_MEMORY is the highest-priority onboarding file.
* Major architecture decisions must be written into GitHub.
* Research, architecture, development, and validation must remain separate.
* 小C does not invent strategies, weights, trading logic, research conclusions, or frameworks.
* All development must follow approved 小P specifications.
