## ADL-001

Decision:

GitHub is the only official knowledge base.

Status:

Accepted

Rationale:

All important architecture decisions, specifications, templates, schemas, and cases must be preserved in the repository. Chat history is not official memory.

---

## ADL-002

Decision:

Task ID convention uses:

* G-xxx Research
* P-xxx Architecture / Review
* C-xxx Development
* V-xxx Validation

Status:

Accepted

Rationale:

The convention keeps ownership and repository history traceable.

---

## ADL-003

Decision:

Project Memory is the highest-priority onboarding file.

Status:

Accepted

Rationale:

Any new ChatGPT session must be able to understand the project quickly without asking the user to repeat long context.

## ADL-004

Decision:

「紀錄」 means Project Sync.

Status:

Accepted

Rationale:

When the user says 紀錄, the project sync files must be updated so the repository remains the official memory.
