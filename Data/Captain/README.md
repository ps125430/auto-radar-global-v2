# Captain Intelligence Inputs

Status: Shadow Runtime Only

本目錄保存 Captain Profile、My Ship 與單一 Active Mission 的人工核准輸入。
所有配置均由資料檔提供，Runtime 不得硬編碼資金比例、持有天數或風險預算。

- `captain_profile.json`: 市場、預算、風險、持有期與資金配置邊界。
- `ship_state_2026-07-01.json`: 可序列化的 My Ship 快照。
- `mission_2026-07-01.json`: 當日唯一 Active Mission。

這些資料不產生選股、交易訊號或訂單。
